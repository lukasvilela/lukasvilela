## 👨‍💻 O que eu faço aqui

Esta é minha conta de trabalho. Os repositórios são **privados** (código proprietário), então aqui fica o **resumo do que construo** — porque o que importa é o que entrega valor, não só o que é público.

Atuo como **desenvolvedor full-stack** numa plataforma de **recrutamento com Inteligência Artificial (ATS)**, do front à infraestrutura, além de ferramentas internas com agentes e LLMs.

### 🤖 IA &amp; LLM em produção
- Análise de **currículos e entrevistas** com Gemini / Vertex AI
- Análise **facial híbrida** (pipeline Python de métricas + narrativa por LLM)
- **Servidores e agentes MCP** (FastMCP + OAuth) expondo APIs internas a assistentes de IA
- Automações **RPA** portadas pra plataforma própria (scheduler, dashboards, relatórios)

### 🔌 Integrações de verdade
- **Stripe** — planos, gateway, webhooks, billing (test + live)
- **LinkedIn API** — publicação e hunting de vagas
- **N8N + WhatsApp (WPPConnect)** — automações e notificações
- **Multi-posting** de vagas em plataformas externas (XML feed, conectores)

### 🚀 Full-stack &amp; Infra
- **React + Material UI** no front · **Node.js + Express** na API
- **PostgreSQL** e **MongoDB** (incluindo migração entre eles, com Prisma)
- **Docker** local · deploy em **Google Cloud Run**
- Mobile com **Capacitor/Android**
